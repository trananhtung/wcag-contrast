# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project

## [0.1.0] - 2026-06-21

### Added

- Initial release.
- `Rgb` with `from_hex` (3/6-digit, case-insensitive) and `relative_luminance`.
- `contrast_ratio` / `contrast_hex` — WCAG 2 contrast ratio between two colors.
- `level` + `WcagLevel` — AA/AAA pass classification for normal and large text.
- Zero dependencies.
